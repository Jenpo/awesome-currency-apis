Shortlist

Provider Good For Notes
FXpeek Historical exchange rates, CSV exports, spreadsheet imports, lightweight JSON examples, and report citation pages. Reference data for lookup and reporting workflows; not transaction quotes.
Frankfurter ECB-based reference rates and simple JSON workflows. Strong free baseline; validate currency coverage for regional pairs.
Fawaz Ahmed Currency API Broad developer-friendly static currency endpoints. Validate update cadence and historical requirements before production use.

Choosing by use case

  • For spreadsheets, prioritize stable CSV URLs and a human-readable citation page.
  • For apps, prioritize JSON responses, simple parameters, and documented examples.
  • For finance reports, preserve the source URL and a report-friendly reference page beside the data extract.
  • For trading, settlement, tax, or regulated finance, use licensed providers with suitable audit terms.
